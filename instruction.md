# Калькулятор дробей.
Виды обыкновенных дробей:
Правильная дробь — это дробь, у которой числитель меньше знаменателя (например,  ). Такие дроби всегда меньше единицы.
Неправильная дробь — это дробь, у которой числитель больше знаменателя или равен ему (например,  ). Такие дроби всегда больше или равны единице.
Смешанное число (или смешанная дробь) — это запись числа, которая содержит в себе и целую часть, и дробную часть (например,  ). Дробная часть смешанного числа, как правило, является правильной дробью.

## Назначение клавиш.
[:.] - {подробнее} - ссылка вниз к разделу инструкции.
[÷] - обозначает горизонтальную черту дроби.
[.:] - {подробнее}.
[√] - {подробнее}.
[x²] - {подробнее}.
[xʸ] - {подробнее}.

Ввод дроби в калькулятор с помощью скобок и разделительной черты:
A(B÷C) - это дробь, в которой целых частей A; числитель B; знаменатель C. Выражение 3(4÷(5-2))  интерпритируется, как  3+(4÷(5-2)), где между целой и дробной частью знак сложения.

Для ввода дроби в калькулятор, используется отделение целой и дробной части с помощью скобок и знак горизонтальной дробной черты ÷. При этом закладывалась логика автодополнения вводимого выражения: 
выражение A(B÷C) - это дробь с целой частью А, поэтому A(B÷C)=A+(B÷C), где  вместо В и С могут быть выражения в скобках;
выражение  A(B+Д)÷C - это дробь с числителем  A*(B+Д) и знаменателем С, аналогично вместо А, В, С и Д могут быть выражения в скобках.

ФИНАЛЬНЫЙ СПИСОК ПРАВИЛ АВТОДОПОЛНЕНИЯ ДЛЯ ШАГ 3. 
ПРАВИЛО 1: СМЕШАННАЯ ДРОБЬ → ВСТАВЛЯЕМ + (БЕЗ ДОПОЛНИТЕЛЬНЫХ СКОБОК)
Условие: число( ... ) где:

Внутри скобок ровно одна операция ÷ на верхнем уровне (не внутри вложенных скобок)

И после закрывающей скобки НЕТ ÷ или /

Результат: число + ( ... ) → без дополнительных скобок

Примеры:

2(3÷6) → 2+(3÷6) → 2(1÷2) (смешанная дробь)

4((4-3)÷3) → 4+((4-3)÷3) → 4(1÷3) (смешанная дробь)

3(7÷(5+2)) → 3+(7÷(5+2)) → 4 (смешанная дробь)

5((5+2)÷(8-3)) → 5+((5+2)÷(8-3)) → смешанная дробь

2((12÷17+1÷3)÷(8+3)) → 2+((12÷17+1÷3)÷(8+3)) → смешанная дробь

2(3÷(3-19)) → 2+(3÷(3-19)) → смешанная дробь

-3(4÷5) → -3+(4÷5) → отрицательная смешанная дробь

ПРАВИЛО 2: СМЕШАННАЯ ДРОБЬ С ОПЕРАТОРОМ → ОБОРАЧИВАЕМ В СКОБКИ
Условие: Перед смешанной дробью есть явный оператор (*, ÷, /, +, -)

Результат: Оператор сохраняется, смешанная дробь ОБЯЗАТЕЛЬНО оборачивается в скобки (число+(...))

Примеры:

2*3(4÷5) → 2*(3+(4÷5))=7(3÷5) → умножение на смешанную дробь

3 * 4(1÷2) → 3 * (4+(1÷2))=13(1÷2) → умножение на смешанную дробь

2*3(4÷5)÷7 → 2*(3+(4÷5))÷7=1(3÷35)→ деление на смешанную дробь

2+3(4÷5) → 2+(3+(4÷5)) → сложение со смешанной дробью

2-3(4÷5) → 2-(3+(4÷5)) → вычитание смешанной дроби

(2÷3)*4(1÷2) → (2÷3)*(4+(1÷2)) → умножение дроби на смешанную

ПРАВИЛО 3: УМНОЖЕНИЕ → ВСТАВЛЯЕМ *
Условие 1: После закрывающей скобки есть ÷ или /

2(3÷6)÷7 → 2+(3÷6)÷7 → деление

2(12÷17+1÷3)÷(8+3) → 2*(12÷17+1÷3)÷(8+3) → деление

4(4-3)÷3 → 4*(4-3)÷3 → деление

Условие 2: Внутри скобок НЕТ ровно одной операции ÷ на верхнем уровне

3(1÷2+1÷2-3÷2) → 3*(1÷2+1÷2-3÷2) → умножение

4(8+5-2) → 4*(8+5-2) → умножение

5(4-3) → 5*(4-3) → умножение

2(3(4÷5)) → 2*(3+(4÷5)) → умножение (вложенная смешанная дробь)

ПРАВИЛО 4: УМНОЖЕНИЕ ДРОБЕЙ → ВСТАВЛЯЕМ *
Условие: Две дроби подряд без оператора

(2÷3)(4÷5) → (2÷3)*(4÷5) → умножение дробей

(2-3)(4÷5) → (2-3)*(4÷5) → умножение

(2÷3)(4+5) → (2÷3)*(4+5) → умножение

ПРАВИЛО 5: СЛОЖНЫЕ ВЫРАЖЕНИЯ С НЕСКОЛЬКИМИ ОПЕРАТОРАМИ
Условие: Внутри скобок есть + или - И ÷ на верхнем уровне (не ровно одна ÷)

3(1÷2+1÷2-3÷2) → 3*(1÷2+1÷2-3÷2) → умножение

2(12÷17+1÷3) → 2*(12÷17+1÷3) → умножение

ФИНАЛЬНЫЙ АЛГОРИТМ (ПОРЯДОК ДЕЙСТВИЙ)
Проверить наличие оператора перед числом (*, ÷, /, +, -):

ДА → применяем Правило 2 (оборачиваем в скобки)

НЕТ → продолжаем

Проверить наличие ÷ после закрывающей скобки:

ДА → вставляем * (деление) → Правило 3

НЕТ → продолжаем

Проверить верхний уровень внутри скобок:

Ровно одна ÷ и нет других операторов → Правило 1 (вставляем +, без скобок)

Иначе → Правило 3 (вставляем *)

Для дробей без числа перед скобкой → Правило 4 (вставляем *)
